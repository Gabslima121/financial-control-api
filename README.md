# financial-api

Estrutura inicial de projeto TypeScript para Node.js.

## Scripts

- `npm run dev`: roda em modo desenvolvimento com recarregamento.
- `npm run build`: compila TypeScript para `dist/`.
- `npm run start`: executa o código compilado de `dist/`.
- `npm run typecheck`: valida tipos sem emitir arquivos.

## Requisitos

- Node.js 18+.

## Como iniciar

1. Instale dependências: `npm install`.
2. Desenvolvimento: `npm run dev`.
3. Build: `npm run build` e depois `npm run start`.
